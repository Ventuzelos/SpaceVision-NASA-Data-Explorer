#Ata_Reunião_SpaceVision_02

##Ordem de Trabalhos

1.	Ponto de situação do projeto.
2.	Revisão das tarefas concluídas.
3.	Estruturação das pastas do frontend.
4.	Finalização do design do logotipo.
5.	Definição das próximas tarefas e responsabilidades.



##Desenvolvimento da Reunião

A reunião iniciou-se com um ponto de situação geral do projeto, onde cada elemento apresentou o trabalho desenvolvido desde a última reunião.
Foi realizada a criação e estruturação das pastas do projeto na parte do frontend, organizando os diretórios de forma a facilitar o desenvolvimento e a manutenção do código nas próximas fases.
Foi também finalizado o design do logotipo do projeto SpaceVision, tendo sido validada a versão final da identidade visual, garantindo uma imagem simples, moderna e adaptada a ambientes digitais.
Durante a reunião foi feita uma verificação das tarefas concluídas, confirmando-se que os objetivos definidos para esta fase foram cumpridos dentro do prazo estabelecido.
Por fim, foram definidas e distribuídas as próximas tarefas entre os elementos do grupo, assegurando a continuidade do desenvolvimento do projeto de acordo com o cronograma previsto.


##Conclusões

A reunião permitiu confirmar que o projeto se encontra a evoluir conforme o planeado, com a estruturação das pastas do frontend concluída e o design do logotipo finalizado. Ficaram definidas as próximas tarefas, bem como os respetivos responsáveis, garantindo a continuidade do desenvolvimento dentro do cronograma estabelecido.
Nada mais havendo a tratar, a reunião foi dada por encerrada.
