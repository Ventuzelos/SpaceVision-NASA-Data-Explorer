# Arquitetura do Projeto

## Objetivo

O projeto SpaceVision foi desenvolvido seguindo uma arquitetura baseada em componentes reutilizáveis, promovendo organização, escalabilidade e facilidade de manutenção.

---

# Estrutura

```text
Pages

↓

Layouts

↓

Sections

↓

Components

↓

Common Components
```

---

## Organização

### assets

Imagens, ícones e recursos gráficos.

### components

Componentes reutilizáveis da interface.

### layouts

Estrutura geral das páginas.

### pages

Páginas da aplicação.

### services

Comunicação com as APIs da NASA.

### styles

Design System e estilos globais.

### hooks

Custom Hooks React.

### utils

Funções auxiliares.

---

## Design

O projeto segue um Design System próprio baseado em:

- Design Tokens
- Componentes reutilizáveis
- Mobile First
- Dark Theme
- CSS organizado por componente

---

## Objetivos

- Código limpo
- Escalabilidade
- Reutilização
- Boa experiência do utilizador
